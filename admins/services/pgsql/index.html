PostgreSQL

L'hôte

Le serveur tourne sur l'hôte panzer.ares (DNS : 10.0.5.3) C'est une machine virtuelle actuellement virtualisée par arme.

Les données

Les données des bases de données sont persistantes et sont stockées sur notre serveur de stockage de données : ronflex

Chemin sur ronflex: /exports/postgresql Chemin sur panzer: /var/lib/postgresql

Le dossier est monté via NFS.
