MySQL

L'hôte

Un serveur SQL tourne sur l'hôte bertha.ares (DNS : 10.0.5.4) C'est une machine virtuelle actuellement virtualisée par arme.

Les données

Les données des bases de données sont persistantes et sont stockées sur notre serveur de stockage de données : ronflex

Chemin sur ronflex: /exports/mysql Chemin sur bertha: /var/lib/mysql

Le dossier est monté via NFS.
